# JavaScript Homework #10

This project contains two JavaScript tasks completed as part of the GoIT Fullstack course.

## Features

### Task 1 – Countdown Timer

A countdown timer built with **Flatpickr**.

<img width="697" height="290" alt="image" src="https://github.com/user-attachments/assets/00a4fc4b-9c81-4ac0-bbb7-ebf82d8b8d2b" />


#### Functionality

- Select a future date and time
- Validate the selected date
- Prevent selecting a past date
- Start the countdown
- Display the remaining:
  - Days
  - Hours
  - Minutes
  - Seconds
- Automatically stop the timer when the countdown reaches zero

#### Technologies

- JavaScript (ES6+)
- Flatpickr

---

### Task 2 – Promise Generator

A promise generator using **iziToast** notifications.

<img width="641" height="372" alt="image" src="https://github.com/user-attachments/assets/4976e5ec-df1f-4de3-b24a-7bbf0c4dd434" />


#### Functionality

- Set a delay in milliseconds
- Choose the Promise state:
  - Fulfilled
  - Rejected
- Create a Promise with the specified delay
- Resolve or reject the Promise after the selected time
- Display the result using iziToast notifications

#### Technologies

- JavaScript (ES6+)
- Promise
- setTimeout
- iziToast

---
